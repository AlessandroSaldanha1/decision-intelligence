# Product Overview — Decision Intelligence

## Visão do Produto

Decision Intelligence é uma plataforma de Inteligência Organizacional que transforma conhecimento disperso em software entregável.

**Pitch:** "Transformamos conhecimento organizacional em software entregável."

## Problema

Times de produto e engenharia perdem tempo reinventando a roda. Decisões técnicas, padrões de implementação, riscos conhecidos e lições aprendidas ficam presos em comentários de tasks, conversas no Slack, e na cabeça das pessoas.

Quando uma nova demanda chega, ninguém sabe:

- Se algo similar já foi feito
- Quais riscos o time já enfrentou nessa área
- Quais decisões técnicas já foram tomadas
- Quanto tempo levou na última vez

## Solução

Decision Intelligence conecta ao ClickUp do time, indexa o conhecimento organizacional via RAG, e usa Claude para:

1. Recuperar contexto relevante do histórico
2. Analisar a nova demanda com esse contexto
3. Gerar artefatos de produto/engenharia de alta qualidade
4. Criar as tasks estruturadas diretamente no ClickUp

## Fluxo Principal

```
ClickUp → Indexação → Nova Demanda → RAG Search → Organizational Insights
→ Claude Analysis → Artefatos → Plano de Entrega → Publicação no ClickUp
```

## Diferencial

Não é apenas um gerador de documentos. É um sistema que reutiliza o conhecimento real do time para melhorar cada nova demanda com base no que já foi aprendido.

## Personas

- **Product Manager**: quer gerar User Stories com mais contexto e menos ambiguidade
- **Tech Lead**: quer que as especificações já venham com decisões técnicas alinhadas ao padrão do time
- **Engenheiro**: quer receber tasks claras com critérios de aceite, DoD e casos de teste
- **QA**: quer ter cenários BDD e casos de teste gerados automaticamente
